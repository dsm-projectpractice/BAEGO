API 에러 상태 UI

Step사용자 동작화면 반응비고
1홈 진입 또는 탭 전환API 응답 4xx/5xx → 에러 아이콘 + 안내 텍스트 + "다시 시도" 버튼 표시스켈레톤 자리에 에러 UI 교체
2"다시 시도" 탭API 재호출 → 스켈레톤 → 카드 렌더링

네트워크 오류

Step사용자 동작화면 반응비고
1네트워크 단절 상태에서 액션 시도"네트워크 연결을 확인해주세요" 토스트 표시하단 3초간 노출 후 자동 소멸
2네트워크 복구 후 재시도정상 API 흐름으로 복귀
device_id 등록(POST /api/users/register) 실패 시: 로컬 임시 ID로 동작 유지, 재시도는 다음 앱 실행 시 수행.
즐겨찾기 토글 API 실패 시: 아이콘 상태 롤백 + 오류 토스트 표시.
