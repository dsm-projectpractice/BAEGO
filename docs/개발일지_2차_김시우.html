* 주요 개발 내용

  1. 위치 감지 커스텀 훅 개발

    - `navigator.geolocation.getCurrentPosition`을 래핑한 `useGeolocation.ts` 작성.

    - 로딩 상태, 에러 객체, 성공 시 위경도(lat, lng) 반환 구조 설계.

    - 앱 어디서나 `useFilterStore`를 통해 추천 조건을 변경할 수 있도록 구조화.


  3. Mock API 연동 테스트

    - 실제 서버 API가 완성되기 전, 백엔드 담당자와 협의한 응답 스펙대로 Mock JSON 작성.

    - Axios를 이용하여 Mock 데이터를 패치하고 UI 컴포넌트에 렌더링하는 흐름 검증 완료.

[스크린샷 1] Zustand를 활용한 필터 전역 상태 관리 Store 코드

import { useState, useEffect } from 'react'; export const useGeolocation = () => { const [location, setLocation] = useState({ lat: 37.4979, lng: 127.0276 }); // Default: 강남역 const [error, setError] = useState(null); useEffect(() => { if (!navigator.geolocation) { setError('Geolocation is not supported'); return; } navigator.geolocation.getCurrentPosition( (position) => { setLocation({ lat: position.coords.latitude, lng: position.coords.longitude }); }, (err) => { console.warn('Geolocation Error:', err.message); setError(err.code === 1 ? 'PERMISSION_DENIED' : 'TIMEOUT'); // 강제 디폴트 위치 할당 유지 }, { timeout: 5000, maximumAge: 60000 } ); }, []); return { location, error }; };

[스크린샷 2] 에러 발생 및 타임아웃 예외 처리가 포함된 useGeolocation 커스텀 훅 구현
이슈 관리 및 해결 과정

* 발생한 이슈 (Issue):

  - 사용자가 모바일 기기나 브라우저에서 위치 정보 접근 권한을 거부(Denied)하거나 타임아웃이 발생할 경우, 앱이 로딩 화면에서 멈춰버리는 현상 발생.


* 원인 분석:

  - `useGeolocation` 내부에서 `getCurrentPosition`의 에러 콜백(`errorCallback`)에 대한 상태 업데이트 및 예외 처리 UI(Fallback)가 구현되지 않아 무한 로딩 상태에 빠짐.


* 해결 과정 및 결과 (Resolution):

  - 에러 콜백 함수 내에 에러 코드(`PERMISSION_DENIED`, `TIMEOUT`)별 분기 처리 추가.

  - 권한 거부 시 서울 중심부(강남역)를 기본(Default) 좌표로 강제 셋업하고, "위치 권한을 허용하지 않아 기본 위치로 추천합니다"라는 Toast 메시지를 띄우도록 수정.

  - API 호출 시 `timeout: 5000` 옵션을 주어 5초 이상 응답이 없으면 에러로 간주하도록 보완함. (결과: 예외 상황에서도 서비스가 멈추지 않고 정상 작동).
KPT 회고 및 차주 계획

* Keep (유지할 점):

  - `useGeolocation`처럼 API 로직을 Custom Hook으로 완전히 분리하여 UI 컴포넌트는 오직 렌더링에만 집중할 수 있게 설계한 점.


* Problem (문제점/아쉬운 점):

  - 백엔드 개발이 진행됨에 따라 API 응답 포맷(특히 페이징 메타데이터)이 초기 명세와 조금씩 달라져, Mock 데이터 구조를 자주 수정해야 하는 번거로움이 발생함.


* Try (시도할 점/개선 방향):

  - 3주차에 진행될 FE/BE API 명세 미팅에서 최종 포맷을 픽스하고, 이후에는 Axios 인터셉터를 활용하여 전역적인 에러 핸들링 구조를 마련.


* 차주(5~6주차) 개발 계획:
